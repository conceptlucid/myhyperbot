## HyperBot Project

Built HyperBot — a Manus-like AI agent for remote computer control.

**Features:**
- Agent with computer control (mouse, keyboard, screen, terminal, files)
- Cloud dashboard (Next.js)
- AI chat (OpenRouter with free models)
- Security: injection detection, rate limiting, command validation
- Install script: `curl -sL hyperbot.sh | bash`

**GitHub:** conceptlucid/HyperBot

**Components:**
- `hyperbot.sh` — Install script
- `hyperbot-agent/` — Agent code
- `hyperbot-cloud/` — Cloud dashboard + API + AI

**Config:**
- OpenRouter API key included (free Llama 3.2 90B model)
- Default model: meta-llama/Llama-3.2-90B-Vision-Free
